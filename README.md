# chef-node-cleaner

## Setup
```bash
git clone git@github.com:evanlok/chef-node-cleaner.git
```
```bash
bundle install
```

Set ENV variables for:
- CHEF_API_ENDPOINT
- CHEF_API_CLIENT
- CHEF_API_KEY
- AWS_ACCESS_KEY_ID
- AWS_SECRET_ACCESS_KEY
- AWS_REGION

## Run
```bash
bundle exec ruby node_cleaner.rb
```
